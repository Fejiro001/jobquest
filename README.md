# JobQuest

![JobQuest Preview](./assets/media/website.png)

## Description

**JobQuest** is a modern job search platform that helps users find employment opportunities based on their location and preferences. The application demonstrates how to integrate real-time job data and location services while providing a clean, accessible, and responsive user experience.

This project focuses on practical job search functionalities, map interactions, and user preference management using modern front-end development practices.
